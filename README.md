# BossWinAgent · 智能求职助手

> 全自动 BOSS 直聘投递 + AI 接管沟通 —— 让你睡觉时也在投简历、跟 HR 聊。

## 下载安装
前往 [**Releases**](../../releases) 下载最新 `BossWinAgent-windows.zip`，解压后运行 `BossWinAgent.exe`，控制台会**直接在程序窗口内打开**（无需另开浏览器）。

---

## 核心优势

### 🎯 招呼语「前20字」杀手锏
> BOSS 招聘端消息预览**只显示前约 20 个字**，HR 很可能不点开看全文。

普通工具：「你好，看到你们在招前端，我很感兴趣…」—— 前 20 字全是废话，HR 直接划走。

BossWinAgent 把**最强钩子前置到前 20 字**，自动读岗位 JD + 你的简历，生成：
> **「5年全栈React+Node，做过医疗影像系统，和你们这岗高度匹配…」**

年限 + 匹配技能 + 匹配度一眼抓住 HR —— **回复率差距巨大**。

### 🤖 AI 全自动接管
- 定时自动投递（时间段 / 间隔 / 每日上限可调）
- HR 回复 **AI 以你本人口吻自动回**，绝不暴露是机器人
- 自动发简历 / 换微信（走 BOSS 官方通道）

### 🧠 智能相关性过滤（只投对口岗）
- 标题黑/白名单：自动跳过销售、客服、运营等无关岗
- **AI 匹配度闸门**：读 JD 对比你简历打分，技术栈不符（C#/.NET/Java 等你没有的）自动跳过
- **自动过滤外包公司**：内置主流外包名单，可自定义

### 🖥️ 真实浏览器 + 可视化
- 用本机 Chrome / Edge / Firefox（自动检测、可切换）
- 投递/沟通窗口内可见，登录态持久

---

## 使用
1. 首次打开**邮箱注册 / 登录**（直接注册即可用）
2. 控制台里设置浏览器、关键词、简历摘要、过滤规则、定时投递
3. 浏览器窗口扫码登录 BOSS，剩下交给它

## 系统要求
- Windows 10 / 11 64 位
- 已装 Chrome 或 Edge（或用内置 Firefox 引擎）

## ⚠️ 重要说明（使用前必读）

- **强制注册登录**：为防范滥用、保障合规，本工具目前**强制注册/登录后才能使用**，请大家理解。
- **测试阶段 · 封号风险未知**：本工具目前**正在测试中**，BOSS 直聘的封号概率/风控表现**尚不明确**，请大家**抱着学习交流的心态使用**，自行评估风险。账号一旦异常请立即停止自动化、回平台手动操作。
