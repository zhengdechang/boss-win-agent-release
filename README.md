# BossWinAgent · 智能求职助手

> 全自动 BOSS 直聘投递 + AI 接管沟通 —— 让你睡觉时也在投简历、跟 HR 聊。
> 一个能搜、能筛、能投、能聊、能换微信的 AI 求职 Agent，全程在你本机浏览器里跑。

## 下载安装
前往 [**Releases**](../../releases) 下载最新 `BossWinAgent-windows.zip`，解压后运行 `BossWinAgent.exe`，控制台**直接在程序窗口内打开**（无需另开浏览器）。

## 核心优势

### 🎯 招呼语「前20字」杀手锏
> BOSS 招聘端消息预览**只显示前约 20 个字**，HR 很可能不点开看全文。

普通工具：「你好，看到你们在招前端，我很感兴趣…」—— 前 20 字全是废话，HR 直接划走。

BossWinAgent 把**最强钩子前置到前 20 字**，自动读岗位 JD + 你的简历，生成：
> **「5年全栈React+Node，做过医疗影像系统，和你们这岗高度匹配…」**

年限 + 匹配技能 + 匹配度一眼抓住 HR —— **回复率差距巨大**。

### 🤖 AI 全自动接管
- 定时自动投递（时间段 / 间隔 / 每日上限可调）
- HR 回复 **AI 以你本人口吻自动回**，绝不暴露是机器人
- 自动发简历 / 换微信 / 留电话（走 BOSS 官方通道）
- 微信式聊天界面，头像、未读角标一目了然

## 它能做什么（完整能力）

### 🔍 搜索 & 投递
- **全国 60+ 城市**搜索，多维筛选：薪资 / 经验 / 学历 / 公司规模 / 融资阶段 / 福利
- **多页翻页扫描 + 批量投递**，带进度、可随时取消
- **公司去重**：模糊匹配公司名变体，同一家只投一次
- **HR 活跃度识别**：自动跳过长期不活跃的招聘者，不浪费招呼次数
- **🎯 智能投递**：按公司分组 → 每家自动挑「最高/最对口的 HR」→ 一键批量投，命中率更高

### 🧠 三大 AI 助手
- **岗位分析**：匹配度打分、关键技能、差距建议、决策(值得投/谨慎/跳过)、风险点、面试建议提问
- **简历优化**：拆解 JD 核心要求、匹配差距、建议关键词、项目改写示例、行动清单（24h 缓存）
- **沟通建议**：开场破冰、聊天话题方向、踩雷避坑、跟进话术、引导面试、沟通风格（24h 缓存）

### 💬 聊天 & 交换
- AI 自动接管 HR 对话，第一人称、自然口语、绝不暴露 AI
- 自动在合适时机交换**微信 / 简历 / 电话**
- 每个会话可单独**暂停/恢复 AI**、手动接管、一键跳转浏览器到聊天页

### 🧭 智能相关性过滤（只投对口岗）
- **标题黑/白名单**：自动跳过销售、客服、运营等无关岗
- **AI 匹配度闸门**：读 JD 对比你简历打分，技术栈不符（C#/.NET/Java 等你没有的）自动跳过
- **自动过滤外包公司**：内置主流外包名单
- **法人直聘识别**：HR 是公司法人/老板本人时特别标记，沟通策略更对路
- 🔧 以上**全部可在设置页自定义**：白/黑名单关键词、AI 匹配分阈值、外包公司名单、薪资门槛，按你的方向随意增删调整

### 📊 数据 & 效率
- **投递转化漏斗**：待投 → 已投 → HR 回复 → 面试，一眼看清转化
- **本地候选清单**：看到心仪岗位先收藏，攒一批再统一投
- AI 简历优化结果可**联动注入招呼语**，让开场更精准
- 真实浏览器引擎：用本机 **Chrome / Edge / Firefox**（自动检测、可切换），登录态持久，投递/沟通窗口内可见

## 使用
1. 首次打开**邮箱注册 / 登录**（直接注册即可用）
2. 进入后**扫码登录 BOSS 直聘**（弹出浏览器，手机 App 扫码）
3. 在设置页配置：AI 平台 / Key、简历摘要、关键词、过滤规则、定时投递
4. 搜索岗位 → 智能投递 / 批量投递，剩下交给它自动跟 HR 聊

## 系统要求
- Windows 10 / 11 64 位
- 已装 Chrome 或 Edge（或用内置 Firefox 引擎）
