# BossWinAgent 下载

BossWinAgent —— Windows 桌面客户端，控制本机浏览器（Chrome / Edge / Firefox）自动投递与沟通。

## 下载安装
前往 [**Releases**](../../releases) 下载最新 `BossWinAgent-windows.zip`，解压后运行 `BossWinAgent.exe`，会自动打开控制台 `http://127.0.0.1:8010`。

## 使用
1. 首次打开需**登录**（账号由管理员发放，无账号无法使用）
2. 登录后在控制台设置浏览器、搜索关键词、定时投递等
3. 浏览器窗口可见，可手动扫码登录 BOSS

## 系统要求
- Windows 10/11 64 位
- 已安装 Chrome 或 Edge（或使用内置 Firefox 引擎）
